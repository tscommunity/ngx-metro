What is it?

Metro 4 is an open source toolkit for developing with HTML, CSS, and JS. This library is a wrapper for this toolkit and brings the beautiful components in your angular application.

Build responsive, mobile-first projects on the web with the first front-end component library in Metro Style and not only.
