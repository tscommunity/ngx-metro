Form wrapper

The m4-form-wrapper is useful to wrap the controls of a form together. It will create an error display for the group and makes the creation of forms a bit easier.

Note that you should use MetroFormGroup when using the m4-form-control-wrapper or m4-form-wrapper.
The MetroFormGroup provides a form name that is used for the generation of error codes.
